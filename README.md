# jandanpic

屏蔽已读的无聊图，每次以打开jandan.net/pic作为更新已读时间戳标准。
当访问到上次的时间戳对应的帖子的时候，把最近一次打开jandan.net/pic对应的时间戳设置为已读。

```
+-----------------------------------------------------+
|                                                     |
|     jandan.net/pic      jandan.net/pic/XXXX         |
|           +-----------------------+                 |
|           |       readable        |   hiden         |
|           |                       |                 |
|   +---+ +---+ +---+ +---+ +---+ +---+ +---+ +---+   |
|   |   | |   | |   | |   | |   | |   | |   | |   |   |
|   |   | |   | |   | |   | |   | |   | |   | |   |   |
|   |   | |   | |   | |   | |   | |   | |   | |   |   |
|   +---+ +---+ +---+ +---+ +---+ +---+ +---+ +---+   |
|                                                     |
+-----------------------------------------------------+
```
